Veranstaltung hinzufügen
Einzeltermin oder Terminserie erstellen
Tipp: Kategorien können direkt bearbeitet werden. Farb-/Namensänderungen werden auf alle Termine dieser Kategorie übernommen. Beim Löschen bleiben Termine erhalten und wechseln automatisch zur Standardkategorie.
Hinweis: „Nur diesen Termin löschen“ erzeugt eine Ausnahme (Ex-Date) für die Serie.
Zeitfenster im Kalender: 05:00–22:00
Termin löschen
—
Bei „Nur diesen Termin“ bleibt die Serie bestehen, aber dieser einzelne Tag wird als Ausnahme gespeichert.
Änderung anwenden
—
Bei „Nur diesen Termin“ bleibt die Serie bestehen (Ausnahme + Einzeltermin). Bei „Ganze Serie“ wird die Regel der Serie geändert.
Termin kopieren
—
Es wird ein neuer Einzeltermin erstellt (unabhängig von Serien). Wochenfilter wird auf Alle Wochen gesetzt.
Tip: Danach kannst du per Drag/Resize anpassen.
Export
Backup (JSON) oder Kalenderdatei (ICS)
JSON enthält alles (eigene Events + ausgeblendete Basisblöcke + Ausnahmen). ICS ist zum Import in externe Kalender gedacht.
Import
JSON (Backup) oder ICS (Kalender)
ICS wird als einzelne Termine importiert (keine Serien-Regeln).
